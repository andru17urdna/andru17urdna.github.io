Waveform

Waveform is a pixel for pixel clone of spotify. Utilizing a flask backend to easily make queries to the SQL Database. Information in the frontend is stored by using Reacts state management to dynamically render data on screen.

Shuffled

Shuffled is an experiment of CSS with a resemblance in concept to Untappd. Shuffled uses express in the backend to fetch and query an SQL database to provide users with information about the current pack of cards they are viewing. Connecting the backend to the front I've used React-Redux to render information on screen.

  • Live Site
  • GitHub

Speak Easy

Speak Easy is currently a work in progress, that aims to be an easy to use small and large event planner. Easily organize events or house partys and invite friends without having to individually message each person. This app combines the ease of use from Flask SQLalchemy and react-redux to allow users to easily monitor events they are attending or created.
